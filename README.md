# Used Car Price Prediction 🚗

This project focuses on predicting the price of used cars using machine learning.

## Description
In the used car market, deciding the right price can be difficult. In this project, I built a machine learning model that estimates the price of a car based on factors like brand, year, fuel type, transmission, and kilometers driven. 

The dataset was cleaned and analyzed to understand the patterns, and different models were tested to find a reliable prediction. The final model can give an approximate price based on user inputs.

## Tools Used
- Python  
- Pandas  
- Scikit-learn
